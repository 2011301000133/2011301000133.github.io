RANSAC算法
优点是模型无视异常点的影响

step1：从数据集中随机抽取样本作为内点集合  
step2：用内点集合来拟合模型  
step3：把剩余数据放入模型，其中一定误差内的样本加入到内点集合  
step4：如果迭代到达预定次数，或者其它结束条件，结束，否则转到step2  


```
from sklearn.linear_model import RANSACRegressor
```
